1. 使用方法
windows 64位：delout.exe 与 chan-code-generator.exe
mac os：命令行执行python delout.py 与 python chan-code-generator.py
目前后场的代码模板是配置好的，chan-code-generator生成代码，delout删除所有生成的代码，out_sv目录下是生成的代码文件


2. 目录文件说明：

out_sv
	生成的代码文件存放目录，可更改，在svfiles.json中配置
templates_sv
	生成代码的模板文件，可更改，在svfiles.json中配置
	目录下文件后缀必须为 .template

keys.txt
	模板文件替换key配置
bo.txt
	bo层代码中属性

svfiles.json
	模板和输出文件配置文件
	filepath 输出文件目录
	filename 输出文件名称，支持以key命名
	template 模板文件

delout.exe
	删除所有生成的代码
chan-code-generator.exe
	生成代码
